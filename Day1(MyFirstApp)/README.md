# Day 1: My First .NET App 🚀

This day marks the beginning of my .NET journey! I set up my development environment and created my very first console application.

## 📝 Key Concepts
- **Project Structure**: Understanding `.csproj`, `bin`, and `obj` folders.
- **Entry Point**: Introduction to `Program.cs`.
- **CLI Commands**: Using `dotnet new`, `dotnet build`, and `dotnet run`.

## 📂 Files
- `Program.cs`: The main entry point of the application.
- `MyFirstApp.csproj`: The project file defining dependencies and settings.

## 🚀 Getting Started
```powershell
dotnet run
```
